# 🎬 JioCinema Clone

A full-stack clone of the JioCinema streaming platform built using **React** (frontend) and **Express.js + Node.js** (backend). Users can search for movies.

---

## 🛠 Tech Stack

### Frontend:
- ⚛️ React
- 🎨 CSS Modules
- 🔍 Movie search functionality

### Backend:
- 🧠 Node.js
- ⚙️ Express.js
- 📦 Custom API serving movie data
